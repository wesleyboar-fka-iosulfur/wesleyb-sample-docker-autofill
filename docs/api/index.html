Schoology Autofill Project - API

This is all the code for the API service logic.

It assumes the Koa Web Framework is used by its consumer, so it exports a Koa middleware that has a routes function.

Requirements

  • Node 10.X+ (do not exceed LTS)
    • NPM 6.X+ (or latest available for given Node version)

Quick Start

  1. Require the file's koaRouter.
  2. Use the routes in the app (or through a middleware).

Example:

const router = require('./api/main.js').koaRouter;

// Use directly
app.use( router.routes());
// Or via middleware
// subdomain.use( 'api', router.routes());
